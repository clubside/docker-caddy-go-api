Docker, Caddy and Go API Single Page App

This is a simple SPA to demonstrate a containerized, secure web development environment.

The project was designed to be a walkthrough of various technologies and each step of this journey is available to download here or from the GitHub repository

The test API features two routes. The first will generate a random key of a length you specify. The second will attempt to scrape a URI and generate a fancy link based on any OpenGraph data on the page.
